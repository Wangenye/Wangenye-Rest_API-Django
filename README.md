


# Rest_API-Django

## To set Up the project
  <ul type="i">
  <h3>Run</h3>
  <li>pip3 install virtualenv</li>
  <li>virtualenv Env</li>
  <li>pip3 install -r requirements.txt</li>
</ul>

### To run The project using Docker 

$ sudo docker-compose build

$ sudo docker-compose up
